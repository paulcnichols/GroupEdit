GroupEdit - Nothing like Collabedit

A Collabedit Knockoff

Collabedit is an online editor that lets people perform live coding. It has a number of features that make it extremely useful for interviewing programmers. It's been pretty slow lately, so I set out to see how difficult it would be to recreate it.

Enjoy!
